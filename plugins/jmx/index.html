JMX

The JMX plugin in hawtio gives a raw view of the underlying JMX metric data, allowing access to the entire JMX domain tree of MBeans.

General Navigation
The JMX Tree

The topmost level of the JMX tree view lists JMX domains.

Click the icon to expand a tree item and further navigate into a domain.

MBeans are generally indicated with while domains and MBeans that have descendants are indicated using .

The JMX Toolbar

The JMX toolbar is used to interact with an MBean.

The next few sections describe the options provided by the JMX plugin.

Attributes

The Attributes view shows a table of attributes for the selected MBean

Data is organized in columns generally following a name/value format.

Operations

The Operations view shows a list of available JMX operations that can be invoked on the selected MBean.

Click the icon to expand the operation invocation form which is dynamically generated from JMX operation metadata.

Fill in the fields with the operation parameters and click the “ Execute” button to invoke the JMX operation. Clicking the “ Reset” button will reset the form, clearing all of the fields. Clicking the “ Cancel” button will reset and close the form. A successful invocation of the JMX operation will be reported in the body of the form.

Chart

The Chart view displays charts of numeric metric values updated in realtime using Cubism.js. As metrics are periodically fetched from the server the charts will scroll from right to left. Hover the mouse over a portion of the chart data to view values from that time period.

When the Chart view is selected from the JMX toolbar an additional “ Edit Chart” option is enabled allowing customization of which JMX metrics are displayed on the chart.

Other Functions

Plugins can extend the JMX toolbar and add additional functions. Descriptions of these functions can be found in the plugins documentation such as ActiveMQ or Camel.
