This page allows you to connect to remote processes which already have a jolokia agent running inside them. You will need to know the host name, port and path of the jolokia agent to be able to connect.

If the process you wish to connect to does not have a jolokia agent inside, please refer to the jolokia documentation for how to add a JVM, servlet or OSGi based agent inside it.

If you are using Fabric8, JBoss Fuse, or Apache ActiveMQ; then a jolokia agent is included by default (use context path of jolokia agent, usually jolokia, for Apache ActiveMQ the context path is api/jolokia). Or you can always just deploy hawtio inside the process (which includes the jolokia agent, use Jolokia servlet mapping inside hawtio context path, usually hawtio/jolokia). The port number is usually 8181 for Fabric8, JBoss Fuse, and 8161 when using Apache ActiveMQ.
