  • demo of FMC and hawtio with Fuse Fabric

Articles on using hawtio

The following articles have been found on the web about hawtio. If you find any others please let us know or raise an issue and we'll add it to the list.

  • Running Apache ActiveMQ and hawtio in Standalone Mode (2016) by Bennet Schulz
  • Hawtio + ActiveMQ by Christian Posta
  • Hawtio & Apache Jclouds by Ioannis Canellos
  • Apache Camel web dashboard with hawtio by Claus Ibsen
  • introducing the Apache Camel based open source iPaaS by James Strachan
  • Monitoring Camel Application by Gnanaguru
  • Hawtio, la console web polyvalente by Gérald Quintana (in French) with a great overview of hawtio (note: Google can translate the page into readable english)
  • Hawtio, écrire un plugin by Gérald Quintana (in French) the second part on writing a plugin (note: Google can translate the page into readable english)
  • Hawtio authentication with LDAP by Jochen Cordes with a blog series how to use LDAP authentication with hawtio running on Tomcat 7, JBoss Fuse, and JBoss EAP.
  • Installing hawtio as web console for ActiveMQ by Dejan Bosanac shows how to install hawtio as the web console in Apache ActiveMQ. Dejan also demonstrates how you can remote connect from hawtio to any ActiveMQ broker.
  • HawtIO on JBoss Wildfly 8.1 — step by step by Christian Posta How to get HawtIO running and secured on JBoss Wildfly Application Server 8.1
  • HawtIO on OpenShift by Jimmi Dyson, how to run Apache Tomcat on OpenShift Online and install hawtio into the Tomcat.
  • Hawtio on JBoss EAP 6 by Jochen Cordes how to deploy Hawtio on JBoss EAP 6
